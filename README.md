## 它到底在做什么事

最简单的一句话：把外语页面变成双语对照，原文在上，译文在下，排版尽量不乱。

听起来简单，但做好其实挺难的。一般的翻译插件要么把整个页面翻掉（读起来感觉少了什么），要么要手动框选复制粘贴（麻烦到懒得用）。沉浸式翻译的思路是保留原文，在旁边或下面补一行译文，所以叫"沉浸式"——你仍然是在读原文，只是多了一个随叫随到的参考。

这个逻辑对以下几类人特别有用：

- **每天要读大量英文资料的人**，包括研究员、从事外贸或互联网行业的人
- **学外语的人**，因为对照着看比单独看翻译学得快
- **追外国视频但字幕不够好的人**，沉浸式翻译支持 60+ 平台的实时双语字幕
